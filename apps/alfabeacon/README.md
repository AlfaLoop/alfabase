# AlfaBeacon


## AlfaBeacon Frame

AlfaBeacon is made up of the following elements

* Apple iBeacon packets
* AlfaBeacon Service uuid:  0xA55A
* AlfaBeacon Service data:

Byte<br/> Offset | Description<br/>
--:|:------:|
0 | Device type | 
 1 | Battery level | 
 2~7 | MAC Address| 
 
 
* Device name: AlfaBeacon (scan response)
